# Profiler

# What are Profiler?

Profiler have been introduced in Release 1.3.0 of SLM. They are used to gather information about devices registered in SLM and publish them via AAS submodels registered in AAS of the respective device. Profiler can be added dynamically via the REST API of the Resource Registry.

# Default Profiler

By default, SLM provides the following Profiler:

  • Ansible Facts (opens new window)
  • Webcams (opens new window)
